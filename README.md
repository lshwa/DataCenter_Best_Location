# AI 하이퍼스케일 데이터센터 최적입지 분석

국내 AI 하이퍼스케일 데이터센터 최적입지 선정을 위한 거시/미시 2단계 공간분석 프로젝트입니다.

## 웹사이트

🔗 **[라이브 데모 보기](https://lshwa.github.io/DataCenter_Best_Location/)**

## 분석 개요

### 거시분석 (지역 단위)
- 전국 144개 시군구 대상
- 전력 여유용량, 재생에너지 잠재량, 용수, 교통, 지가 등 평가
- **Top 15 후보 선정**

### 미시분석 (250m 격자 단위)
- 하드 필터 10종 적용 (법적 규제, 자연보전, 재해 위험)
- 소프트 스코어 6종: 변전소거리 15% / 교통 20% / 경사도 25% / 냉각수 20% / 산업단지 15% / 민원 5%
- **BFS + Spectral Graph Clustering 2단계 파이프라인**

## 클러스터링 방법론

```
BFS (물리적 연결) → Spectral (균질 핵심 구역 선별)
```

1. **BFS**: 인접 375m 내 격자 연결 → 연속 배치 가능한 물리적 구역 정의
2. **Spectral**: BFS 구역 내 가우시안 유사도 행렬 + eigengap → 균질 핵심 구역 선별

## 기술 스택

- **공간분석**: GeoPandas, Shapely, Rasterio (EPSG:5179)
- **클러스터링**: SciPy Sparse, scikit-learn Spectral/KMeans
- **데이터**: OSM Overpass API, VWORLD WFS, 국가공간정보포털 SHP
- **시각화**: Leaflet.js, Chart.js, contextily, Folium
